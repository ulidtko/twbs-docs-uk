Як використовувати

Для кращої продуктивності, всі значки потребують базовий клас та індивідуальний клас значків. Розмістіть наступний код будь-де. Не забудьте поставити пробіл між значком та текстом для правильного відступу (padding).

Не змішуйте з іншими компонентами

Класи значків не можуть поєднуватись з іншими елементами. Вони спроектовані, щоб бути окремими елементами.

<span class="glyphicon glyphicon-search"></span>

Приклади

Використовуйте їх у кнопках, групах кнопок для панелі інструментів, панелі навігації, або в приставках елементів форми.

<button type="button" class="btn btn-default btn-lg">
  <span class="glyphicon glyphicon-star"></span> Star
</button>

Спадаюче меню

Готовірозкриватись, контекстні меню для відображення списку лінків. Їхня інтерактивність забезпечується JavaScript плагіном для спадаючого меню.

Приклад

Обгорніть кнопку спадаючого меню та список у ньому за допомогою класу .dropdown, або іншого елементу, котрий оголошує position: relative;.

  • Інша дія
  • Щось іще
  • Відокремлений лінк
<div class="dropdown">
  <button class="btn dropdown-toggle" type="button" id="dropdownMenu1" data-toggle="dropdown">
    Dropdown
    <span class="caret"></span>
  </button>
  <ul class="dropdown-menu" role="menu" aria-labelledby="dropdownMenu1">
    <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Дія</a></li>
    <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Інша дія</a></li>
    <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Щось іще</a></li>
    <li role="presentation" class="divider"></li>
    <li role="presentation"><a role="menuitem" tabindex="-1" href="#">Відокремлений лінк</a></li>
  </ul>
</div>

Опції вирівнювання

Додайте .dropdown-menu-right до .dropdown-menu для вирівнювання спадаючого меню праворуч.

Застаріле .pull-right вирівнювання

У версії 3.1, ми визначили застарілим вирівнювання .pull-right для спадаючого меню. Для вирівнювання меню праворуч, використовуйте .dropdown-menu-right. Компоненти, вирівняні праворуч у навігаційній панелі, використовують mixin версію класу right-aligned, для автоматичного вирівнювання меню. Для перевизначення їх, використовуйте .dropdown-menu-left.

<ul class="dropdown-menu dropdown-menu-right" role="menu" aria-labelledby="dLabel">
  ...
</ul>

Групи кнопок

Згрупуйте серію кнопок разом в одну лінію, створивши таким чином групову кнопку. За допомогою JavaScript та нашого плагіну кнопок, можна ще додати стиль поведінки перемикачів або галочок (checkbox).

Підказки та інформери в групі кнопок потребують спеціальних налаштувань

При використанні підказок або інформерів (popovers) в межах .btn-group, необхідно визначити опцію container: 'body', щоб уникнути небажаних ефектів (наприклад, коли елемент стає ширшим та/або коли втрачається закругленння кутів при виклику підказок чи інформерів).

Основний приклад

Обгорніть серію кнопок з класом .btn в клас .btn-group.

<div class="btn-group">
  <button type="button" class="btn btn-default">Ліва</button>
  <button type="button" class="btn btn-default">Середня</button>
  <button type="button" class="btn btn-default">Права</button>
</div>

Групи форми введення

Розширте елементи управління форми додавши текст або кнопки до, після, чи з обох сторін будь-якого текстового поля форми. Застосовуйте .input-group з .input-group-addon для приставки елементів спереду чи позаду .form-control.

Крос-браузерна сумісність

В даному випадку уникайте використання елементів <select>, оскільки вони в повній мірі не можуть стилізуватись у WebKit браузерах.

Підказки та інформери в групі введення потребують спеціальних налаштувань

Коли використовуєте підказки або інформери (popovers) на елементах у межах .input-group, вам необхідно визначити опцію container: 'body', для уникнення небажаних побічних ефектів (наприклад, коли елемент стає ширшим та/або коли втрачається закругленння кутів при виклику підказок чи інформерів).

Навігаційна панель за промовчанням

Налаштування контрольних точок

Залежність від плагіну

Чутлива навігаційна панель вимагає у вашій версії Bootstrap плагіну для переключення.

Доступність навігаційних панелей

Переконайтесь що додали role="navigation" до кожної навігаційної панелі для покращення їх доступності.

Форми

Розмістіть вміст форми в межах .navbar-form для правильного вертикального вирівнювання, та для забезпечення функцією згортання на вузьких оглядових вікнах. Використовуйте параметри вирівнювання, щоб вирішити, де буде знаходитись вміст у межах навігаційної панелі.

Як провідний, клас .navbar-form має більшість свого коду спільно з .form-inline через mixin.

Бренд 
<form class="navbar-form navbar-left" role="search">
  <div class="form-group">
    <input type="text" class="form-control" placeholder="Пошук">
  </div>
  <button type="submit" class="btn btn-default">Відправити</button>
</form>

Застереження для мобільних пристроїв

Є деякі застереження щодо використання елементів управління форми всередині фіксованих елементів на мобільних пристроях. Прогляньте нашу документацію підтримки браузерів для уточнення.

Завжди додавайте ярлики

Програми читання екранів (screen readers) будуть мати проблеми з вашими формами, якщо ви не будете додавати ярлики для кожного поля введення. Для даних вбудованих форм навігаційних панелей, ви можете приховати ярлики використовуючи клас .sr-only.

Кнопки

Додайте клас .navbar-btn до елементів <button>, котрі не знаходяться у <form> для вертикального їх центрування в навігаційних панелях.

Бренд 
<button type="button" class="btn btn-default navbar-btn">Ввійти</button>

Специфічна особливість використання

Як і стандартні класи кнопок, .navbar-btn може використовуватись в елементах <a> та <input>. Тим не менше, жоден .navbar-btn або ж стандартні класи кнопки не повинні використовуватись в елементах <a> в середені .navbar-nav.

Текст

Обгорніть рядок тексту в елемент з класом .navbar-text, зазвичай з тегом <p>, для правильного вирівнювання та кольору.

Бренд

Signed in as Mark Otto

<p class="navbar-text">Signed in as Mark Otto</p>

Неконтекстні лінки

Люди, котрі використовують стандартні лінки в межах незвичайних компонентів навігаційних панелей, можуть застосовувати клас .navbar-link для додавання правильних кольорів у навігаційних панелях за промовчанням та інвертованих навігаційних панелях.

Бренд

Signed in as Mark Otto

Список груп

Список груп є гнучким та потужним компонентом для відображення не лише простих пунктів списку, але і для чогось складнішого, що стосується користувальницького вмісту.

Основний приклад

Самим базовим є простий список групи з ненумерованими пунктами, та власними класами. Додавайте до нього опції, як це показано нижче, або при необхідності додавайте власні стилі CSS.

Панелі

Хоча не завжди необхідно, але іноді потрібно в щось упакувати DOM. Для таких випадків, спробуйте компонент панелі.

Основний приклад

За промовчанням, всі .panel застосовують деяку базову рамку (border) та відступи (padding), щоб вміщувати певний вміст.
